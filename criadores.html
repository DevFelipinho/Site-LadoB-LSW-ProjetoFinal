Criadores

Meu nome é Felipe Lara Facin, sou estudante do curso de Análise e Desenvolvimento de Sistemas do Instituto Federal do Triângulo Mineiro (IFTM). Este site foi desenvolvido como parte de um projeto da disciplina de Fundamentos de Web Design, com o objetivo de colocar em prática os conhecimentos adquiridos em sala de aula, integrando programação e design para criar um site funcional.

Durante o desenvolvimento, explorei linguagens de programação como HTML e CSS, além de conceitos de UX/UI. Também contei com a ajuda de minha namorada, Mariana Rocha Soares de Souza, para a pesquisa de produtos do site e para o design.

O objetivo principal é facilitar as compras pra quem gosta de antiguidades musicais, criando uma plataforma única para esse nicho.
